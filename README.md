# README

This is a ROR + React project for managing student information

## Versions

* Ruby - 2.7.0
* Rails - 6.0.2
* React - 16.13.1
* Node - 14.6.0

## Development

* Rails unit testing - Rspec
* Rails serializer - Fast JSON Serializer
* HMR (Hot module replacement)
* SASS
* SQLite DB
* ESLINT enabled
* Flow enabled
* Create PR and merge
* React routing (Lazy loading) with chunking
